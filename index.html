Map of Armed and Unarmed Police Shootings

This map shows the locations of police shootings involving armed and unarmed individuals. Police shootings of unarmed suspects have attracted greater public scruitiny recently, opening conversations on police brutality and accountability. Armed shootings are denoted by a red handgun icon while unarmed shootings are denoted by a blue empty palm icon. Information such as the date of the incident, the race of the individual involved, whether the individual survived, and a summary report are provided by clicking on each individual map marker.

Future Vision

This project is a very early draft. If there was more time, better icons and images would be used to better contextualize the information provided on the map. There would be less reliance on modal text popups and more focus on visual cues such as color, age, etc. Data trends would also be visualized using graphs to compare statistics and features such as varaible filters would be implemented. But for now, enjoy this simple presentation of armed and unarmed police shootings across the United States.

This map was created using data collected from Deadspin Police Shooting Database
